Sobre el sistema


Objetivo del Sistema

Este es un proyecto de facultad de código abierto y gratuito, creado para ayudar a personas con discapacidades funcionales. El objetivo es ofrecer un sistema de facturación fácil de usar y diseñado para que sea más comprensible y accesible, facilitando el proceso de facturación en el mundo de los negocios modernos.


Motivación para hacer el sistema

El mundo laboral aún no está completamente preparado para incluir a personas con discapacidades. Con este sistema, quiero aportar mi ayuda para que puedan gestionar sus propios emprendimientos de manera más sencilla. Mi objetivo es hacer que el proceso de facturación sea más fácil de entender y realizar, reduciendo el tiempo y esfuerzo necesario para poder facturar.


Cómo está armado el sistema

El sistema está desarrollado utilizando Python, HTML, CSS, JavaScript y lenguaje de fórmulas de Excel.
