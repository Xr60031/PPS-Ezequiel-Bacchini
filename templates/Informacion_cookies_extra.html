Cookies del sistema

¿Que son las Cookies?

Las cookies son pequeños archivos que guarda una página web en tu computadora o celular para recordar cosas sobre vos


¿Como el sistema usa las cookies?

Este sitio web utiliza cookies para guardar algunos datos importantes mientras lo estás usando. Estos datos incluyen:

  • • Tu nombre y el nombre de tu empresa.

  • • Tu número de CUIT.

  • • Información técnica como tu certificado digital, llave privada y ticket de acceso. (Esto no es tu clave fiscal)

  • • El nombre del archivo Excel que estás usando y su identificador.

  • • Algunos indicadores internos para que el sistema funcione correctamente.


¿Para que guarda el sistema estas cookies?

Estos datos se guardan únicamente para que puedas usar todas las funciones del sistema de forma segura y sin errores.

Esta información no se comparte con terceros para fines comerciales ni publicitarios. Algunos datos, como el CUIT o certificados digitales, pueden ser utilizados exclusivamente para cumplir con los requisitos de la AFIP en el proceso de facturación. Toda la información se utiliza únicamente durante tu sesión, con el objetivo de que el sistema funcione correctamente y facilitar tu trabajo.
